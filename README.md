# Sri Padmavati Venkateswara Swami Temple

A modern, responsive website for Sri Padmavati Venkateswara Swami Temple, built with React, TypeScript, and Tailwind CSS.

##  Project Overview

This is the official digital presence of Sri Padmavati Venkateswara Swami Temple, providing professional services for temple information, darshan timings, events, and community engagement.

##  Technology Stack

- **Framework**: React 18.3+
- **Language**: TypeScript 5.8+ (Strict mode)
- **Styling**: Tailwind CSS 3.4+ with custom theming
- **Build Tool**: Vite 5.4+
- **UI Components**: shadcn/ui with Radix UI primitives
- **Routing**: React Router v6
- **Animation**: Framer Motion 12+
- **State Management**: React Query 5.8+
- **Forms**: React Hook Form with Zod validation

##  Getting Started

### Installation

npm install

### Development

npm run dev

### Build

npm run build

##  Type Safety

Strict TypeScript configuration with full type checking enabled, no implicit any, strict null checks, and unused variable detection.

##  Responsive Design

Fully responsive: Mobile (320px+), Tablet (640px+), Desktop (1024px+)

##  Accessibility

WCAG 2.1 Level AA compliant with semantic HTML, ARIA labels, and keyboard navigation support.

##  Performance

- Image lazy loading
- Code splitting
- Tree-shaking with ES modules
- Optimized bundle with Vite

##  Deployment

Deploy to Vercel, Netlify, GitHub Pages, or AWS S3 + CloudFront
